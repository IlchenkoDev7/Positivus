# WEBPACK

## Подготовка к запуску
1. Установка проекта из репозитория
2. Установка зависимостей
```bash
npm i
```


## Команды

### Запуск сервера для разработки
```bash
npm run start
```

### Сборка проекта без оптимизации
```bash
npm run build-dev
```

### Сборка проекта с оптимизацией
```bash
npm run build-prod
```

### Очистка папки dist
```bash
npm run clear
```
